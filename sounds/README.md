# Sounds

This directory contains audio files used by agent-sims-hook.sh for status feedback.

## Required Sound Files

Copy your sound files here:
- working.wav (or .mp3) - Played when Claude starts working
- thinking.wav (or .mp3) - Played when Claude is thinking
- offline.wav (or .mp3) - Played when Claude goes offline

## Usage

The agent-sims-hook.sh script plays these sounds to give audio feedback on Claude's status.

Manually copy your preferred sound files to this directory after cloning the repo.
